# Reference Files

Documentation and reference materials from the original Framer export.
These files are NOT used in the build - they exist for understanding what Framer generates.

## Files

### framer-ssr.css
Complete SSR (Server-Side Rendered) styles extracted from Framer's output.
Contains:
- Color tokens (`--token-*` custom properties)
- Base resets and typography
- All `.framer-*` class definitions
- Responsive breakpoint styles

Useful for:
- Looking up original Framer class definitions
- Understanding token values when migrating to semantic CSS
- Reference when debugging style differences
